#  Project Overview

This project analyzes Swiggy food order data using Microsoft Excel to uncover sales trends, customer preferences, restaurant performance, and regional demand patterns.

The objective was to transform raw order-level data into an interactive dashboard that enables stakeholders to monitor key business metrics and identify growth opportunities.

Using Excel functions, Pivot Tables, Pivot Charts, Maps, and Slicers, the dashboard provides a comprehensive view of sales performance across states, cities, food types, and time periods.



#  Business Problem

Food delivery platforms generate thousands of orders daily, making it difficult to manually track performance and customer behavior.

The goal of this project was to answer key business questions such as:

- How much revenue is being generated?
- Which cities contribute the most sales?
- What are the sales trends over time?
- How do customer ratings compare across orders?
- Which food type generates more revenue?
- Which states perform best?
- How does sales performance vary by month, week, and day?

The dashboard provides actionable insights to support business decision-making and performance monitoring.



#  Data Preparation

The raw dataset contained the following fields:

- State
- City
- Order Date
- Restaurant Name
- Location
- Category
- Dish Name
- Price
- Rating
- Rating Count

To improve analysis, additional columns were created using Excel formulas:

### Day

Used to analyze weekday vs weekend performance.


### Quarter

Used for quarterly performance analysis.



### Week

Created to track weekly sales trends and identify peak periods.

### Food Type (Veg / Non-Veg)

Classified dishes based on keywords found in dish names.

#  Key Insights

- Total sales exceeded ₹53 million.
- Average customer rating remained above 4.3.
- Vegetarian dishes generated more revenue than Non-Veg dishes.
- Weekend sales significantly outperformed weekdays.
- Bengaluru was the highest revenue-generating city.
- Revenue was concentrated in a few key states and cities.
- Sales remained relatively stable across weeks and months.

# Business Recommendations

- Expand vegetarian menu offerings and promotional campaigns, as Veg items contribute the majority of total sales.
- Increase marketing efforts and operational capacity during weekends to capitalize on higher customer demand.
- Focus customer acquisition and retention strategies in top-performing cities such as Bengaluru, Hyderabad, Mumbai, Lucknow, and New Delhi.
- Investigate low-performing states and implement localized promotions to improve regional sales performance.
- Promote highly-rated restaurants and dishes to leverage strong customer satisfaction and drive repeat orders.
- Introduce combo meals, bundle offers, and cart-value incentives to increase average order value.
- Use weekly and monthly sales trends for better demand forecasting, staffing, and inventory planning.
- Strengthen partnerships with high-performing restaurants to improve customer experience and maximize revenue growth.
